## Welcome to Better Blur DX

Better Blur DX is a fork of the Plasma 6 blur effect with additional features and bug fixes. It extends KWin's blur capabilities with force blur, customizable contrast parameters, corner radius adjustments, and refraction effects.

## Supported Versions

Better Blur DX supports **Plasma 6.5** and **6.6**. The effect is built for the exact KWin version installed on your system and needs to be rebuilt after Plasma upgrades.

<Note>
  X11 support is available but requires explicit build configuration with `-DBETTERBLUR_X11=ON`. The X11 version is not tested as extensively as the Wayland version.
</Note>
